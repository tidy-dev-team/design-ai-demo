# AI-Ready-Storybook

## How to use?
1. Download and install GitHub Desktop ([Link](https://desktop.github.com/download/))
2. Open it and Sign in with your account
3. Clicking the green "<> Code ▾" button
4. From the dropdown select "Open with GitHub Desktop"
5. Follow the prompts to clone repo
6. open this folder in terminal
7. run `sudo npm run storybook`
8. Enter your MacBook user password
9. Wait for Storybook to open in your browser

## How to open this folder in Terminal?
1. In GitHub Desktop app click "Show in Finder"
2. When the folder opens, hold down the `Alt` key
3. This should make a path breadcrumbs appear at the bottom
4. Right click on the last crumb (the folder should be "AI-Ready-Storybook")
5. Select "Open in Terminal"
6. Type "claude" in Terminal window
7. wait 2 seconds for it to load
